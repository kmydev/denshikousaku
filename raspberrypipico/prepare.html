このページでは、Raspberry Pi Pico を使った開発を始めるための準備手順を説明します。 主に Windows 11 の PC と MicroPython を用いて進めていきます。

1. 必要なもの

  • Windows 11 を搭載したパソコン
  • Raspberry Pi Pico H（ピンヘッダ付き）
  • ブレッドボード
  • USBケーブル（データ通信対応）

2. 開発環境：Thonny のインストール

プログラムの作成と実行には、初心者に優しいPython統合開発環境「Thonny」を使用します。 以下のリンクから最新版をダウンロードしてインストールしてください。

▶ Thonny 公式サイト

3. Pico の接続

  • Raspberry Pi Pico をブレッドボードに差し込みます。
  • Pico に USB ケーブルを挿します。
  • Pico の BOOTSEL ボタンを押しながら パソコンに接続します。
  • ボタンを押しながらパソコンに接続すると、Windows に「RPI-RP2」というUSBストレージとして認識されます。

4. MicroPython のインストール

「RPI-RP2」として認識された Pico に、MicroPython のファームウェア（.uf2ファイル）をドラッグ＆ドロップすることでインストールできます。

最新の MicroPython ファームウェアは以下からダウンロードできます：
▶ MicroPython ダウンロード（Pico用）

5. Thonny で接続・確認

  • Thonny を起動します。
  • 「ツール」＞「設定」＞「インタプリタ」タブから「MicroPython (Raspberry Pi Pico)」を選択。
  • シリアルポートが自動的に検出されるか、COMポートを選択します。
  • 設定後、「REPL」ウィンドウに `>>>` が表示されれば接続成功です。

6. プログラムを書いて動かす

下記のような簡単なプログラムを Thonny に入力し、Pico 上で実行してみましょう： 

from machine import Pin
import time

led = Pin(25, Pin.OUT)

while True:
    led.toggle()
    time.sleep(0.5)

LED が 0.5 秒ごとに点滅すれば成功です。

7. 動かないときの確認ポイント

  • Pico に正しく MicroPython がインストールされているか
  • Thonny のインタプリタ設定が正しいか
  • USBケーブルが充電専用ではなく「データ通信対応」であるか
  • 接続時に BOOTSEL ボタンを押したか
  • シリアルポート（COM）が正しく選ばれているか
