Raspberry Pi Pico について

← セクショントップに戻る

Raspberry Pi Pico は、イギリスの Raspberry Pi 財団が開発した小型で安価なマイコンボードです。 独自設計の RP2040 チップを搭載し、デジタル工作やプログラミング学習に最適なデバイスとして人気があります。


特徴としては、デュアルコアの Arm Cortex-M0+ プロセッサ、豊富な GPIO ピン（26本）、低消費電力、そして安価（1,000円前後から）で入手できる点が挙げられます。


プログラミングには主に MicroPython や C/C++ を使用します。USB 経由でパソコンと接続し、初心者でも扱いやすい構成になっています。


Raspberry Pi Picoにはいくつかの種類がありますが、このサイトではRaspberry Pi Pico Hを使って電子部品を制御したり、センサーからデータを読み取ったりする方法を、具体例とともに紹介していきます。
